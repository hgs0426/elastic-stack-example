# hot-warm-architecture

## requirement

- 12GB Memory 

## start containers
``` bash
./run-containers.sh
```

## stop & remove containers
``` bash
./rm
```

## kibana
- http://localhost:5601

## cerebro
- http://localhost:9000

## elasticsearch
- in docker network: http://coordinator-01:9200
- from container outside: http://localhost:9200
